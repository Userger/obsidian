 - [!] значение по умолчанию изменяемый объект может привести к ошибкам.
```python
>>> def buggy(arg, result=[]):
...     result.append(arg)
...     print(result)
...
>>> buggy('a')
['a']
>>> buggy('b')
['a', 'b']
>>>
```
- [i] значение по умолчанию создается в момент определения функции, а не в момент вызова.

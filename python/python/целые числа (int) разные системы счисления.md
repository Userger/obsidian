- `0b1` и `0b0` - двоичная система счисления (binary)
- `0xF` - число в шестнацетиричной системе счисления
	15
- `0o7` - число в восьмиричной системе счисления
	7

# Функции bin, oct,hex

- ## bin(int_number)
```python
>>> bin(55)
'0b110111'
```
- ## oct(int_number)
```python
>>> oct(55)
'0o5'
```
- ## hex(int_number)
```python
>>> hex(55)
'0xf'
```
